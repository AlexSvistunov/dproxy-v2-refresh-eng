Справочник API

*Полная версия доступна в десктопной версии сайта

  • Смена IP

    GET
    /changeip/
    <TOKEN>
    Base URL: http://api.dproxy.co
    Response HTTP/1.1 200 OK
    {"status": "success", "message": "IP was changed"}

  • Установка таймера на смену IP

    POST
    /api/changeip/auto/
    <TOKEN>
    Base URL: http://api.dproxy.co
    Header: Content-Type: application/json
    Body: {"interval_seconds": "60"}
    Response HTTP/1.1 200 OK
    {"status": "success", "message": "IP rotation scheduled: interval 60 sec"}

  • Отмена таймера на смену IP

    POST
    /api/changeip/
    <TOKEN>
    Base URL: http://api.dproxy.co
    Header: Content-Type: application/json
    Body: {"interval_seconds": "0"}
    Response HTTP/1.1 200 OK
    {"status": "success", "message": "IP rotation canceled"}

  • Перезагрузка устройства

    GET
    /api/reboot/
    <TOKEN>
    Base URL: http://api.dproxy.co
    Response HTTP/1.1 200 OK
    {"reboot": "OK", "message": "Reboot is started, wait 1 minute."}

  • Проверка статуса устройства

    GET
    /api/device_status/
    <TOKEN>
    Base URL: http://api.dproxy.co
    Response HTTP/1.1 200 OK
    {"status": "OK", "message": "Device is ready."}
